  <name>Task 2: Add struct instantiation and defer visitors</name>
  <files>src/compiler/compiler.h, src/compiler/compiler.cpp</files>
  <action>
    In compiler.h, add visitor declarations:
    void visitStructInstantiationExpr(StructInstantiationExpr* expr) override;
    void visitDeferStmt(DeferStmt* stmt) override;
    
    In compiler.cpp, implement:
    
    visitStructInstantiationExpr():
    - Push struct name as string constant: chunk.writeString(expr->name, 0)
    - For each field in expr->fields: field.value->accept(*this)
    - Emit OP_NEW_STRUCT with name index and field count:
      emitByte(Opcode::OP_NEW_STRUCT)
      emitByte(static_cast<uint8_t>(nameIndex))
      emitByte(static_cast<uint8_t>(expr->fields.size()))
    
    visitDeferStmt():
    - Compile the expression: stmt->expression->accept(*this)
    - Emit OP_DEFER: emitByte(Opcode::OP_DEFER)
    
    (Note: OP_NEW_STRUCT has 2 operand bytes: name index + field count)
  </action>
  <verify>cd build && cmake .. && make -j$(nproc) 2>&1 | grep -q "Built target hs"</verify>
  <done>Struct instantiation and defer compile to correct bytecode</done>
</task>

  <files>src/compiler/compiler.h, src/compiler/compiler.cpp</files>
  <action>
    In compiler.h, add visitor declarations:
    void visitPanicExpr(PanicExpr* expr) override;
    void visitRecoverExpr(RecoverExpr* expr) override;
    
    In compiler.cpp, implement:
    
    visitPanicExpr():
    - Compile the panic value: expr->value->accept(*this)
    - Emit OP_PANIC: emitByte(Opcode::OP_PANIC)
    
    visitRecoverExpr():
    - Emit OP_RECOVER: emitByte(Opcode::OP_RECOVER)
    
    Modify visitDotExpr():
    - Change from OP_GET_PROPERTY to OP_GET_FIELD
    - Push field name as string constant
    - Emit OP_GET_FIELD instead of OP_GET_PROPERTY
    
    Before (current):
    void Compiler::visitDotExpr(DotExpr* expr) {
        expr->object->accept(*this);
        chunk.writeString(expr->property, 0);
        emitByte(Opcode::OP_GET_PROPERTY);
    }
    
    After:
    void Compiler::visitDotExpr(DotExpr* expr) {
        expr->object->accept(*this);
        int nameIndex = chunk.addStringConstant(expr->property);
        emitByte(Opcode::OP_GET_FIELD);
        emitByte(static_cast<uint8_t>(nameIndex));
    }
    
    (OP_GET_FIELD takes 1 operand byte: the name index)
  </action>
  <verify>cd build && cmake .. && make -j$(nproc) 2>&1 | grep -q "Built target hs"</verify>
  <done>Panic, recover compile correctly; dot expression uses OP_GET_FIELD</done>
