# Phase 13: Type Annotations - GOAL

## Goal Statement
Add proper type annotations on variables and functions, optional chaining (`?.`), and nullish coalescing (`??`) to make Harvis Script more expressive and safe for real-world usage.

## Success Metrics
- Type annotations parse correctly and store type info
- Optional chaining short-circuits on null/undefined
- Nullish coalescing returns left operand if not null/undefined
- All existing tests continue to pass

## Out of Scope
- Runtime type checking
- Type inference
- Type narrowing
- Interface type checking
